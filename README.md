## 💫 About Me

I am a **Frontend Developer** dedicated to building modern, inclusive, and performant web applications. My approach blends technical precision with a deep focus on user experience.

- 🚀 **Core Focus**: Crafting responsive interfaces with React, Next.js, and TypeScript.
- ♿ **Accessibility First**: Committed to WCAG 2.1 standards to ensure the web is usable for everyone.
- 🎨 **Design Driven**: Bridging the gap between design and code using Tailwind CSS and Figma.
- 📱 **Mobile Ready**: Experience in cross-platform development with React Native.
